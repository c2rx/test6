# 一、按下操作面板上的逆变关闭按钮；  

1.1 点击、唤醒屏幕；  

1.2 点击屏幕“操作”按钮，弹出操作菜单；  

![](images/485d9e294fbdbe30465a621bac312852ffa5e53220e7ee8d47c9826ec2b93acd.jpg)  

1.4 若单机逆变关机，选择“本机”，若并机逆变关机选择“并机”；  

![](images/391f285602dadb25f37ab94dba2315c20a6ef01e63ccc5ed7d97cac50d8c9aca.jpg)  

# 二、按下EPO 按钮；  

2.1 按下“EPO”按钮，停止整流器、静态开关和电池的运行。  

![](images/64ed613caf3b29d23f0ef19c358752c9de8537d48d7fa69befc6a002934806e7.jpg)  

# 三、断开外置电池开关；  

3.1 将外置电池总开关由“ON”按压 至“OFF”位置；   
3.2 将外置电池各分开关由“ON”按 压至“OFF”位置。  

![](images/5c29d7678572388d5c61d70a51570b31f66fccfce2b2dfcd03649a4775c79072.jpg)  

4.1 断开主路输入开关、旁路输入开关、输出开关。此时所有内部电源关闭，LCD 显示关闭。  

# 四、依次断开UPS 主路输入开关、旁路输入开关、输出开关。
